# Proyecto gesvitalpro

## Descripción
Este es el proyecto gesvitalpro, una aplicación desarrollada en Laravel para gestionar cirugías y pacientes.

## Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu_usuario/gesvitalpro.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd gesvitalpro
   ```
3. Instala las dependencias:
   ```bash
   composer install
   ```

## Uso
Para iniciar el servidor de desarrollo, ejecuta:
```bash
php artisan serve
```
